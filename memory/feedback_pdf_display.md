name: PDF display approach
description: User does not want PDF parsing/extraction — just embed a full PDF viewer in the app
type: feedback
---

Do NOT parse or extract text/questions from the PDF. Instead, embed a full PDF viewer so the user sees the entire PDF as-is.

**Why:** User wants to show the whole PDF without complex parsing logic.

**How to apply:** Use PDF.js or an iframe-based viewer to render the PDF inline. No text extraction needed for displaying questions.
